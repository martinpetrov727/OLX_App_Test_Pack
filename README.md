# OLX App – Manual Test Cases

## 📌 Overview
This project contains a comprehensive set of manual test cases for the OLX mobile application.  
The goal is to validate core functionalities of the platform and ensure a smooth and reliable user experience.

---

## 🎯 Scope
The test cases cover the following key features:

- Creating listings
- Searching and filtering listings
- Favorites functionality
- Messaging between users
- Notifications
- User interactions with listings

---

## 🧪 Test Coverage
The test suite includes:

- ✅ Positive scenarios (valid user actions)
- ❌ Negative scenarios (invalid inputs and error handling)
- 🔢 Boundary value testing (input limits for title, description, etc.)
- ⚠️ Validation checks (required fields, formats)
- 🔍 Search and filtering logic
- 💬 Messaging functionality
- ⭐ Favorites management

---

## 🛠️ Testing Approach
- Manual testing
- Structured test case design
- Focus on real-world user behavior
- Validation of UI responses and system messages

---

## 📂 Test Case Structure
Each test case includes:

- Test Case ID
- Title
- Preconditions
- Steps
- Expected Result
